About This Site

Slashcode is the site for All Things Slash. Slash is the source code and database that was originally used to create Slashdot, and has now been released under the GNU General Public License. It is a bona fide Open Source / Free Software project. Use this site to get the Slash source, read the latest Slash news, and participate in Slash discussions.

Slashcode is run by the good folks at OSTG, but like all good open source projects, depends on the community. Thanks for your continued support, and let us know what we can do to make the site better.

Slash and Slashcode could not be possible without the use of some great Open Source software.

  • Debian and Red Hat Linux
  • Perl
  • Apache
  • mod_perl
  • MySQL

Slash was originally written by CmdrTaco and CowboyNeal. Slashcode is run by CaptTofu, Krow and pudge.
